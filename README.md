#Learn

To run:

	npm install
	gulp

To generate screenshots of example pages you'll need phantomjs installed and the following packages:

	"node-phantom": "git+https://github.com/williamparry/node-phantom.git",
    "mkdirp": "^0.5.0",
    "ncp": "^2.0.0",
    "serve-static": "^1.9.2",
    "finalhandler": "^0.3.4",
    "glob": "^5.0.3",
    "figures": "^1.3.5"

Then run:

	node screenshots
